Restaurant Landing Page
Project Submission – InternsForge Front-End Internship

1. Project Overview
   This project is a single-page restaurant landing website designed using HTML, CSS, Bootstrap and minimal JavaScript.
   The website contains multiple sections including Hero, About, Menu, Gallery, Testimonials, Contact and Footer.
   The objective of the project is to demonstrate the ability to build a clean, responsive and well-structured front-end layout.

2. Technologies Used
   HTML5 – Structure
   CSS3 – Styling
   Bootstrap 5 – Layout and components
   Bootstrap Icons – Social icons
   Google Fonts – Typography
   Google Maps Embed – Location display

4. Features

Hero Section
A full-screen section containing a background image and call-to-action buttons.

About Section
A two-column layout with a description and an image.

Menu Section
Three food items displayed using Bootstrap cards with uniform image sizes.

Gallery Section
A grid of six images with hover zoom effect.

Testimonials Section
Three customer reviews with profile images and star rating display.

Contact Section
A Google Map embed, contact information and a contact form layout.

Footer
Contains quick links and social media icons on a dark background.
